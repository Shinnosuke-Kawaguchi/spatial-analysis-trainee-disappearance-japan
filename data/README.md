# Data Documentation

このディレクトリには、GitHub 公開用に集計・匿名化した分析入力データのみを置きます。明細データ、作業用ジオコーディングファイル、Word 文書は Git 管理対象外です。

## Files

### `final.xlsx`

都道府県×年の分析用パネルデータです。

- 観測単位: 都道府県 year
- 対象期間: 2020-2024 年
- 観測数: 235 行、47 都道府県 × 5 年
- 用途: 回帰分析、記述統計、空間分析の主入力

### `Citylatlongi.xlsx`

都道府県庁所在地の緯度経度です。

- 観測単位: 都道府県
- 用途: 空間重み行列の作成
- 注意: 作業用の `data/geocoding.xlsx` は公開対象外です。

### `N03-20260101_GML/`

Queen 型の隣接行列を作成するために使用する行政区域データです。
ファイルサイズが大きいため、このリポジトリには含めません。

- 出典: 国土交通省「国土数値情報（行政区域データ）」
- 使用ファイル: `N03-20260101_prefecture.shp`
- 配置場所: リポジトリ直下の `N03-20260101_GML/`
- 注意: `src/final.R` の空間分析を実行する場合は、国土数値情報から同データを取得し、上記パスに展開してください。

## Main Data Sources

`final.xlsx` は、公開統計・公開資料を都道府県×年単位に集計した派生データです。主な出典カテゴリは次の通りです。

- 技能実習生数・失踪者数: 出入国在留管理庁などの技能実習生・失踪者関連の公表統計
- 外国人比率・総人口: 住民基本台帳、在留外国人統計、または e-Stat 系の都道府県別公表統計
- 有効求人倍率: 厚生労働省・都道府県労働局の一般職業紹介状況
- 産業構成: 技能実習計画または受入関連統計から、農林水産、建設、製造の構成比を再集計
- 最低賃金: 厚生労働省の地域別最低賃金
- 日本語教室数: 文化庁の日本語教育実態調査等の公表資料
- 可住地面積: 国土交通省、国土地理院、統計局等の都道府県別公表値
- 行政区域ポリゴン: 国土交通省「国土数値情報（行政区域データ）」
- 労働関連違反件数: 技能実習実施者・受入企業に関する監督指導、公表事案、違反関連資料を都道府県×年で集計
- 緯度経度: 都道府県庁所在地をジオコーディングし、公開用に都道府県レベルへ整理

公開前には、論文・報告書本文で使う正式な出典名、表番号、URL、取得日を人間が再確認してください。

## Variable Definitions

| Variable | Definition |
| --- | --- |
| `Prefecture` | 都道府県名 |
| `Year` | 年 |
| `foreigner_ratio` | 都道府県人口に対する外国人住民等の比率 |
| `job_offer_ratio_avg` | 年平均の有効求人倍率 |
| `disappear_ratio` | 技能実習生数に対する失踪者数の比率 |
| `trainees_total` | 技能実習生数 |
| `disappear_per1000` | 技能実習生 1,000 人あたり失踪者数 |
| `disappeared` | 失踪者数 |
| `total_total` | 人口または関連分母として用いた都道府県別総数 |
| `all_total` | 外国人比率等の計算に用いた総数 |
| `share_primary` | 技能実習受入または関連計画における第一次産業系の構成比 |
| `share_construction` | 建設業の構成比 |
| `share_manufacturing` | 製造業の構成比 |
| `min_salary` | 地域別最低賃金 |
| `jp_school_num` | 日本語教室数 |
| `Inhabitable_area` | 可住地面積 |
| `violation_count` | 労働関連違反件数の都道府県×年集計 |
| `jp_school_density_area` | `jp_school_num / Inhabitable_area`。スクリプトで作成 |
| `ln_min_salary` | `log(min_salary)`。空間モデル用にスクリプトで作成 |

## Processing Policy

- 分析スクリプトは `data/final.xlsx` を主入力とし、明細データから再構築しません。
- 比率変数は、公開用データ作成段階で都道府県×年に集計済みです。
- 日本語教室密度と最低賃金の対数は、再現性のためスクリプト側で作成します。
- 空間重み行列は `Citylatlongi.xlsx` の都道府県庁所在地座標から作成します。
- Queen 型の空間重み行列を作成する場合は、国土数値情報の行政区域データをローカルに配置して使用します。
- 生成物は `results/` または `figures/` に出力し、明細データを再出力しません。

## Publication Notes

- `docs/*.docx`、`data/geocoding.xlsx`、違反明細、raw データは Git に戻さないでください。
- `violation_count` は観測された違反・公表・監督の結果であり、実際の違反発生率そのものではありません。
- 都道府県集計データのため、個人・企業が特定される解釈や記述は避けてください。
- 出典表記、利用条件、再配布可否は公開前に必ず原典で確認してください。
